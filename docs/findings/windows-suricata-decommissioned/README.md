# Windows Suricata Per-Host Saga (Decommissioned)

These scripts handled the per-host Suricata installation on Windows
agents during the v1 architecture. Documented in the thesis QA as
the source of multiple findings (Set-Content BOM bug, Npcap silent
install hang, NPF device path bypass for non-interactive crash).

Per-host Suricata was decommissioned in May 2026 when NIDS was
centralized on pfSense as part of the v2 architecture pivot.
Kept as evidence of engineering work and findings.
